关键关系解读

1. 外部输入路径

InputHandler 不直接改对象字段，而是把玩家输入写入 world.commandQueue，随后由 CommandBus 在 Tick 的命令阶段统一处理。

2. 规则执行路径

MainScene 推进 tick，TickRunner 按 phase 顺序运行各 feature system；这些系统主要读写 World、GameMap、ObjectPool 和地图域管理器。

3. 表现层读取路径

RenderSync、UIManager、DebugOverlay 从 World / GameMap / PresentationState 读取数据做表现，遵循文档里的 Simulation / Presentation 分离。
