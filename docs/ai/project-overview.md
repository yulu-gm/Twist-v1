# 项目概览

## 基本设定

`Twist_V1` 是一款后末日聚落模拟游戏。玩家需要指挥多个幸存者，在充满威胁的环境中建立、扩张并维持自己的基地。

## 核心玩法支柱

- 选择落脚地点
- 建造并扩张可防御的基地
- 管理稀缺资源与环境威胁
- 协调多个具备半自主、目标驱动行为的角色
- 通过系统之间的相互作用生成涌现式故事

## 早期架构目标

- 基于格子的世界表示
- 角色需求与状态模型
- 目标选择与工作分配循环
- 行动执行流水线
- 确定性的模拟 Tick
- 用于可视化与交互的 Phaser 场景层
