# better-ai-doc

当用户手动调用此 skill 时，你需要对 `docs/ai` 目录下的所有文档进行文本润色。请严格执行并遵守以下规则：

1. **保持原意，不增不减**：禁止删除或增加原文没有的信息，只调整文本的措辞、句式等内容，使其更加通顺、专业。
2. **纯净中文**：禁止在中文中穿插英文名词，除非是不可替代的专有名词。
3. **完整表达，禁止缩写**：禁止缩写中文词汇。例如：“根本原因”不能写为“根因”，“格子中心”不能写为“格心”。必须使用完整的词汇表达。
4. **绝对隔离上下文**：禁止阅读 `docs/ai` 文档路径之外的任何文件。你不需要、也绝对禁止去理解这些文档在具体业务中的上下文。你只需要专注于调整目标路径内文档的文本编写方式。

**执行步骤**：
1. 使用工具查找并读取 `docs/ai` 目录下的所有文档（如 `.md` 文件）。
2. 严格按照上述 4 条规则，逐个对文档内容进行文本润色。
3. 将润色后的内容写回原文件。
