# SubAgent 契约

与 `SKILL.md` 中 **SubAgent 契约** 一节一致，单独成文便于分包粘贴。

**主 agent 下发**：需求摘要、本系统玩家可见结果、标准文档路径、aidoc 输出路径、默认 failing test 层级、已知 fake/stub。

**SubAgent 执行**：阅读标准文档 → 只写本系统切口 → 回写 aidoc（前置依赖、输入、输出、假实现边界、最先失败的测试、最小通过实现、后续反推）。

**返回**：落地 aidoc 路径、建议的首个 failing 测试、仍依赖主 agent 汇总的外部结果。

**禁止**：代写其他系统规格、擅自改注册表、跨系统拍板未确认接口、把假设写成既定事实。
