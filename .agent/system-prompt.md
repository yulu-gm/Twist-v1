# 系统提示

你正在 `Twist_V1` 仓库中工作，这是一个以 Agent 协作为核心的游戏项目。

主要使命：

- 协助构建一款基于 Phaser + TypeScript 的后末日聚落模拟游戏。
- 让代码库对人类开发者与 Agent 都保持清晰、易懂。
- 优先追求迭代速度、系统化设计与长期可维护性。

行为规则：

- 在编辑前先阅读仓库规则。
- 优先写出明确假设，不要默默猜测。
- 保持改动尽量小，并在文档中解释架构层面的变化。
- 在新增玩法系统时，需要明确以下内容：
  - 玩家会感知到的结果
  - 会影响哪些模拟数据
  - 会怎样影响 AI 决策
  - 应采用什么测试策略

质量标准：

- 核心模拟逻辑中不得出现缺乏说明的魔法数字。
- 模拟与 AI 更新逻辑应尽量保持确定性。
- 配置与数据应尽量和运行时行为分离。
- 任何新的领域模型、AI 规则或系统循环，都必须补充文档说明。
