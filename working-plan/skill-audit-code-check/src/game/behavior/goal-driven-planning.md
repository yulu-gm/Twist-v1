# 审计报告: src/game/behavior/goal-driven-planning.ts

## 1. 漏做需求 (Missing Requirements)

- **[指控]** 目标得分未纳入「当前携带状态」。`needScoreForGoal` / `buildGoalDecisionCandidate` 仅使用 `pawn.needs`、曼哈顿距离修正与夜间睡眠乘数；`ChooseGoalPlannerInput` 也未扩展携带物或工单上下文，无法在「手上有物资需先放下 / 优先搬运」等情形下调整需求 vs 闲逛的排序。
- **[依据]** `oh-code-design/行为系统.yaml` 中「行动评分器」职责：综合「紧急度、距离、**当前携带状态**、时间语境」。

- **[指控]** 休息目标未体现「前往**自己的**床铺」的归属约束。候选床铺仅通过 `interactionPointsByKind(..., "bed")` 与 `isInteractionPointReservedByOther` 过滤，未结合小人—床铺分配关系；若网格上存在多张床，可能选中非归属床（除非数据层把非本小人床全部排除，但本文件未表达该规则）。
- **[依据]** `oh-code-design/行为系统.yaml` 关键流程「夜晚休息」：行为系统重新评估疲劳与床铺可用性后，「小人前往**自己的床**并进入休息状态」；`oh-gen-doc/需求系统.yaml` 休息行为：「寻找**属于自己的**床铺」。

- **[指控]** 「漫无目的散步」的实现与策划描述的移动方式不一致。`allReachableWanderPaths` / `chooseWanderPath` 在全图可行走格上反复求路径并偏好更长路径，再随机抽取；而非「随机选择**相邻**可行走格」的一步式闲逛。
- **[依据]** `oh-gen-doc/行为系统.yaml` 中「漫无目的散步」行为特征：`移动方式: 随机选择相邻地图格移动`。

- **[指控]** `nextStepFromPath` 声明接收 `logicalCellsByPawnId` 却完全未使用（参数名为 `_logicalCellsByPawnId`），无法在「沿缓存路径迈一步」时结合他小人占用或动态阻挡做校验；与地图侧「可达与邻接查询」联合决策的意图若存在，则此处为未完成能力。
- **[依据]** `oh-code-design/地图系统.yaml` 关键流程「移动查询」：地图网格与占用管理器联合返回可达候选；行为规划在消费路径缓存时若应与此一致，则本函数当前未接入占用维度。

- **[补充说明]** `GoalKind` 仅含 `eat` | `sleep` | `recreate` | `wander`，**工作类**目标由 `sim-loop` 等路径单独处理。若架构上认定本文件只负责「需求驱动 + 闲逛」子评分器，则不与「工作」缺失直接冲突；但与 `oh-code-design` 中「行动评分器」对「工作行动、需求行动、散步行动」**统一排序**的表述相比，需在体系层明确分工，否则易产生「设计上的单一评分器」与「代码多轨决策」的验收歧义。

## 2. 无用兼容与 Mock (Useless Compatibility & Mocks)

- 未发现 `mock`、`temp`、`// TODO` 或明显仅为兼容废弃入口的死分支，**未发现典型无用兼容代码**。
- **[说明]** `nextStepFromPath` 对 `logicalCellsByPawnId` 的保留形参属于「对外 API 看似支持多小人逻辑、实现未消费」的表面完整性与实现落差，易误导调用方以为已考虑他小人位置；更宜归类为接口债务而非传统 Mock。

## 3. 架构违规 (Architecture Violations)

- **[指控]** 在「需求候选筛选」与「闲逛选路」中直接、高频调用 `findPathAStar`（含对多个交互点逐一规划、以及 `allReachableWanderPaths` 对几乎全图的 A*），使行为规划模块深度承担**路径求解**职责，加重行为层体量。
- **[依据]** `oh-code-design/行为系统.yaml` 风险条款：「若行为系统同时承担**路径**、规则计算、动画驱动，职责会过重」；`oh-code-design/地图系统.yaml` 空间规则层职责含「提供矩形选区与**路径相关基础查询**」，路径能力更贴近地图子系统边界。

- **[说明]** 本文件未直接修改世界状态、未依赖 UI，主要为纯函数式规划；**未发现**「表现层改写核心数据」类越权，主要问题为**路径职责与地图分层**的耦合方式与设计风险陈述不一致。
